Logic Design

Syllabus

  1. BOOLEAN ALGEBRA
    • Introduction, Basic Definitions, Axiomatic Definition and Boolean Algebra
    • Basic Theorems and Properties, Boolean Functions, Canonical Standard forms, Digital Logic Gates.
    • SIMPLIFICATION OF BOOLEAN FUNCTIONS: The Map method , Two- and Three- Variable Maps, Four- Variable Maps, Five- and Six- Variable Maps
    • Product of Sums Simplification, Don't Care Conditions, The Tabulation Method, Determination of PrimeImplicants, Selection of Prime-Implicants.
    • SLE: Mapping Method : 5 and 6 Variables.
  2. COMBINATIONAL LOGIC
    • Introduction, Design Procedure
    • Adders, Subtractors, Code Conversion
    • Analysis Procedure, Multilevel NAND circuits, Multilevel NOR circuits, Exclusive-or and Equivalence Functions.
    • SLE: Design of Combinational Circuit.
  3. COMBINATIONAL LOGIC WITH MSI AND LSI
    • Introduction, Binary parallel adder, Decimal adder
    • Magnitude Comparator, Decoders, Multiplexers
    • Read-Only Memory (ROM), Programmable Logic Array(PLA).
    • SLE: Applications of PLA.
  1. SEQUENTIAL LOGIC
    • Introduction, Flip- Flops, Triggering of Flip Flops
    • Analysis of Clocked Sequential Circuits, State Reduction and Assignment
    • Flip-Flops Excitation Tables
    • Design Procedure, Design of Counters, Design with State Equations
    • SLE: State Reduction and Assignment of Sequential Circuits.
  2. REGISTERS, COUNTERS AND THE MEMORY UNIT
    • Introduction, Registers, Shift Registers
    • Ripple-Counters, Synchronous Counters
    • Timing Sequences, The Memory Unit.
    • SLE: Design of Synchronous Counters.
  3. REGISTER TRANSFER LOGIC
    • Introduction, Inter register Transfer
    • Arithmetic Logic and Shift Micro-Operations
    • Conditional Control Statements, Arithmetic Shifts
    • Instruction Codes, Design of Simple Computer.
    • SLE: Design of Simple Computer.
